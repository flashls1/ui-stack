# SaaS Landing Page Prompt

Generate a modern SaaS product landing page with high visual impact:

## Hero Section

### Desktop
- Full viewport height hero with animated gradient background
- Large headline with gradient text effect
- Subheadline in muted text
- Two CTAs: Primary gradient button (Get Started), Secondary outline button (Learn More)
- Product screenshot/mockup floating with subtle shadow
- Floating decorative elements (glowing orbs, grid pattern)

### Mobile
- Reduced hero height (80vh)
- Stack headline, subheadline, CTAs vertically
- Product image below fold
- Simpler background (static gradient)

## Sections (in order)

1. **Logo Cloud** - Trusted by companies (grayscale logos, subtle)
2. **Features Grid** - 3x2 grid of feature cards with icons
3. **Product Demo** - Large screenshot with glassmorphism frame
4. **Testimonials** - Carousel of customer quotes
5. **Pricing** - 3 pricing cards (Basic, Pro highlighted, Enterprise)
6. **CTA Section** - Final conversion section with gradient background
7. **Footer** - Links, social icons, newsletter signup

## Visual Effects

### Hero
- Animated gradient mesh background
- Floating particles or grid animation
- Glow effect behind product image
- Text reveal animation on load

### Cards
- Glass variant with subtle border
- Hover: lift + enhanced glow
- Spotlight effect following cursor

### Buttons
- Gradient primary buttons
- Glow pulse on hover
- Ripple effect on click

## Typography

- Hero headline: 4xl-6xl, bold, gradient or white
- Section titles: 3xl, semibold
- Body text: base-lg, normal weight, muted color
- All caps labels with letter-spacing for categories

## Responsive Behavior

- Navbar: Glass, sticky, collapses to hamburger on mobile
- Feature grid: 3 cols → 2 cols → 1 col
- Pricing cards: Horizontal → Stacked
- Testimonials: Carousel → Stack

## Interactions

- Navbar transparency changes on scroll
- Sections fade in on scroll
- Stats counter animation
- Smooth scroll to sections
